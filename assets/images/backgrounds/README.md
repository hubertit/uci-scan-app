# Background Assets

This folder contains background images and patterns for the app.

## Types:
- **Splash Backgrounds**: Full-screen backgrounds for splash screen
- **Login Backgrounds**: Subtle backgrounds for login screen
- **Pattern Overlays**: Subtle patterns for UI elements
- **Gradient Images**: Pre-rendered gradients for performance

## Recommended Sizes:
- **Mobile Portrait**: 1080x1920px
- **Mobile Landscape**: 1920x1080px
- **Tablet Portrait**: 1536x2048px
- **Tablet Landscape**: 2048x1536px

## Formats:
- PNG (for images with transparency)
- JPG (for photos and complex images)
- WebP (for optimized web delivery)

## Naming Convention:
- `splash_bg_light.jpg` - Light theme splash background
- `login_bg_pattern.png` - Login screen pattern
- `gradient_primary.png` - Primary color gradient
