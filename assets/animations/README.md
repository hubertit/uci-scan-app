# Animation Assets

This folder contains Lottie animations and other animation files.

## Types:
- **Loading Animations**: Spinners and loading indicators
- **Success Animations**: Check marks and success states
- **Error Animations**: Error states and warnings
- **Onboarding Animations**: App introduction animations
- **Micro-interactions**: Button animations and transitions

## Formats:
- **Lottie JSON**: For complex vector animations
- **GIF**: For simple animations (use sparingly)
- **APNG**: For high-quality animations with transparency

## Recommended Sizes:
- **Small Animations**: 100x100px
- **Medium Animations**: 200x200px
- **Large Animations**: 400x400px

## Naming Convention:
- `loading_spinner.json` - Loading animation
- `success_check.json` - Success animation
- `error_warning.json` - Error animation
- `scanning_pulse.json` - QR scanning animation
- `splash_logo.json` - Splash screen animation

## Performance Tips:
- Keep animations under 2MB
- Use Lottie for complex animations
- Optimize for 60fps
- Test on lower-end devices
