# Illustration Assets

This folder contains illustrations and graphics for the app.

## Types:
- **Onboarding Illustrations**: Graphics for app introduction
- **Empty State Illustrations**: Graphics for empty screens
- **Error Illustrations**: Graphics for error states
- **Success Illustrations**: Graphics for success states
- **Feature Illustrations**: Graphics explaining app features

## Recommended Sizes:
- **Large Illustrations**: 400x400px
- **Medium Illustrations**: 200x200px
- **Small Illustrations**: 100x100px

## Formats:
- SVG (recommended for scalable illustrations)
- PNG (for complex illustrations with transparency)
- Lottie JSON (for animated illustrations)

## Naming Convention:
- `scanning_illustration.svg` - QR scanning illustration
- `empty_tickets.svg` - Empty state for no tickets
- `error_network.svg` - Network error illustration
- `success_check.svg` - Success state illustration
