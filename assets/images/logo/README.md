# Logo Assets

This folder contains the UCI Kigali logo assets in various formats and sizes.

## Recommended Sizes:
- **App Icon**: 1024x1024px (for app stores)
- **Splash Logo**: 512x512px (for splash screen)
- **Header Logo**: 200x200px (for app headers)
- **Small Logo**: 64x64px (for small UI elements)

## Formats:
- PNG (recommended for logos with transparency)
- SVG (for scalable vector graphics)
- JPG (for photos without transparency)

## Naming Convention:
- `uci_logo_primary.png` - Main logo
- `uci_logo_white.png` - White version for dark backgrounds
- `uci_logo_icon.png` - Icon-only version
- `uci_logo_horizontal.png` - Horizontal layout version
