# UI Icon Assets

This folder contains custom UI icons for the app.

## Types:
- **Navigation Icons**: Icons for bottom navigation
- **Action Icons**: Icons for buttons and actions
- **Status Icons**: Icons for different states
- **Feature Icons**: Icons representing app features

## Recommended Sizes:
- **Small Icons**: 16x16px
- **Medium Icons**: 24x24px
- **Large Icons**: 32x32px
- **Extra Large Icons**: 48x48px

## Formats:
- SVG (recommended for scalable icons)
- PNG (for complex icons with transparency)

## Naming Convention:
- `scan_qr.svg` - QR scanning icon
- `ticket_valid.svg` - Valid ticket icon
- `ticket_invalid.svg` - Invalid ticket icon
- `history.svg` - History icon
- `settings.svg` - Settings icon
- `logout.svg` - Logout icon

## Design Guidelines:
- Use consistent stroke width (2px recommended)
- Follow Material Design icon guidelines
- Use UCI Kigali brand colors when appropriate
- Ensure icons are recognizable at small sizes
